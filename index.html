Project On Console

Microsoft Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.

The console.log () is a function in JavaScript which is used to print any kind of variables defined before in it or to just print any message that needs to be displayed to the user. Parameters: It accepts a parameter which can be an array, an object or any message. Return value: It returns the value of the parameter given.
